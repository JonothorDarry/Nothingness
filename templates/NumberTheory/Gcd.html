First problem statement and definitions

Let's say \(q (q \le 10^6)\) queries are given: what is the greatest common divisor of numbers \(w\) and \(v\) where \(1 \le w, v \le 10^9\)?

To answer this question, several definitions and notational nuissances shall be formulated:

  1. Divisor \(d\) of number \(x\) is such number, often denoted by \(d|x\), such that
    \(d|x \iff \exists_{k \in Z}dk=x\)
    If \(d|x\), then \(x\) is divisible by \(d\).
  2. Greatest common divisor (gcd): A number \(g \in N^+\) is gcd of numbers \(w, v\) if and only if:
    \(g=gcd(w, v) \iff g|w \land g|v \land !\exists_h (h|w \land h|v \land h>g)\)
  3. Remainder from dividing number \(x\) by number \(m\), denoted later by \(x\%m\), is equal to:
    \(m \gt x\%m=x-\lfloor \frac{x}{m} \rfloor m \ge 0\)
  4. Two numbers \(a\), \(b\) are equivalent modulo m, if:
    \(a \equiv b \pmod m \iff \exists_{k \in Z} a-km=b\)
    It's worth noting, that \(x \equiv x\%m \pmod m\).

In other words, if \(g\)=\(gcd(w, v)\), then \(g\) divides \(w\) and \(v\), and there is no greater number \(h\) that divides both \(w\) and \(v\). First solution not being absolute brute force (checking all numbers \(x \in \lt 2;min(w,v) \gt \) whether they're gcd - works in \(O(q*min(w,v))\)) is discussed below:

Distribution of all divisors of a number and their upper estimate

If \(d\) is a divisor of \(x\), \(d \ge 1\), then \(d \le \sqrt{x}\) or \( \frac{x}{d} \le \sqrt{x}\); proof by contradiction is trivial: if \(d|x\) then, there exists such natural \(b\), that \(bd=x\); if \(d \gt \sqrt{x} \) and \(b \gt \sqrt{x}\), then \( db \gt \sqrt{x}\sqrt{x}=x\), which finishes the proof. Furthermore, this shows, that for \(x \ge 1\) occurs \(|\{d:d|x \land d \ge 1\}| \le 2\sqrt{x}\) - because for each divisor \(d\) there is exactly one divisor \(k\) such that \(dk=x\), and \(d \le \sqrt{x} \lor k \le \sqrt{x}\).

First solution

Without loss of generality, let's assume that \(w \lt v\). One can check all positive natural numbers \(k \le w\) whether \(k|w\) and \(k|v\), and take largest such number. To find all divisors \(k: k|w\) one need to check only divisors \(k \le \sqrt{w}\), and check, whether \(\frac{w}{k}|v \lor k|v\), and take maximum of those - this will work in \(O(q*\sqrt{w})\).

Second Solution

Let's create arrays \(a_i\) and \(b_i\) such that \(a_0=w\), \(b_0=v\). One can define these arrays for \(i > 0\) as \(a_i=b_{i-1}\) and \(b_i=a_{i-1}\%b_{i-1}\). When \(b_i=0\) then \(a_i\) is the solution.

Euclidean algorithm

Input format: Two numbers \(w, v\), specifying values, for which \(gcd(w,v)\) has to be calculated.
\(w, v\)

Proofs

Correctness proof

Thesis: for each \(i,\ gcd(a_i, b_i)\) is equal to gcd(\(w\), \(v\)).
Proof by induction: For \(i=0\) \(gcd(a_i, b_i)=gcd(w, v)\), because \(a_0=w, b_0=v\). So: \(\exists_i gcd(a_i, b_i)=gcd(w, v)\). To finish proof is to show that for any \(i\) either \(gcd(a_i, b_i)=gcd(a_{i+1}, b_{i+1})\), or \(b_i=0\)
What can be said about any divisor \(g\) dividing both \(a_i\) and \(b_i\)? If \(a_i\) is shown as \(sb_i +m\), where \(s \in N\), then:
\(a_i = gk, b_i =gl, a_i=sb_i +m\)
\(gk=sgl+m\)
\(gk-sgl=g*(k-sl)=m\)
and so, any common divisor \(g\) dividing both \(a_i\), \(b_i\) has to divide \(a_i\%b_i\) - and so, \(gcd(a_i, b_i)=gcd(a_i\%b_i, b_i)=gcd(b_i, a_i\%b_i)=gcd(a_{i+1}, b_{i+1})\). If \(b_i\) becomes equal to \(0\), then any number divides \(0\), and so any number that is divisor of both \(w\) and \(v\) must divide \(a_i\). The only thing left to do is finding greatest divisor of a number \(a_i > 0\) - which is equal to \(a_i\), which finishes the proof. Furthermore, it shows, that every divisor of both \(w, v\) is also a divisor of \(gcd(w,v)\)

Complexity proof

It can be assumed without loss of generality, that \(a_i > b_i\) - it is true after first iteration. Furthermore,
\(b_i + a_i\%b_i < b_i+b_i=2b_i\)
To prove that \(\frac{a_i + b_i}{a_{i+1}+b_{i+1}} ≥ \frac{3}{2}\) , let's consider two cases: first, when \(a_i ≥ 2b_i\), second, when \(2b_i > a_i ≥ b_i\).
(1) \(\frac{a_i+b_i}{a_{i+1}+b_{i+1}} = \frac{a_i+b_i}{b_i+a_i\%b_i} > \frac{a_i+b_i}{2b_i} ≥ \frac{2b_i+b_i}{2b_i} = \frac{3b_i}{2b_i} = \frac{3}{2}\)
(2) \(\frac{a_i+b_i}{a_{i+1}+b_{i+1}} = \frac{a_i+b_i}{b_i+a_i\%b_i} = \frac{a_i+b_i}{b_i+a_i-b_i} = \frac{a_i+b_i}{a_i} = 1+\frac{b_i}{a_i} > 1+\frac{b_i}{2b_i} = \frac{3b_i}{2b_i} = \frac{3}{2}\)
And so, maximum number of operations can be bounded by \(O(q*log(min(w, v)))\), because sum \(a_i+b_i\) is always greater than 0, after second iteration \(b_2 \lt a_2 \le min(w,v)\), and after \(k\) iterations it becomes lower at least \((\frac{3}{2})^{k}\) times.

Second Problem

Let's say that sole \(gcd(w,v)\) is not enough: You want to find such numbers \(x, y\) such that: \(wx+vy=gcd(w,v)\). One can use same mechanism to solve this problem, but calculate also arrays \(p, q\), such that for each \(i\) following stands:
\(p_iw+q_iv=b_i\)
\(p_1=0, q_1=1 \implies wp_1+vq_1=v=b_1\)
for \(i > 1\) following stands: \(b_i = a_{i-1}\%b_{i-1} = a_{i-1}-\lfloor\frac{a_{i-1}}{b_{i-1}}\rfloor b_{i-1}\)
 It is possble to show \(b_{i-1}\) as \(b_{i-1}=p_{i-1}w+q_{i-1}v\). But what with \(a_{i-1}\)? It's equal to \(b_{i-2}\), except when \(i=2\)... unless \(b_0=a_1=w\). Let also, for simplicity, \(z_i=\lfloor \frac{a_i}{b_i} \rfloor\). Then:
\(p_0=1, q_0=0 \implies wp_0+vq_0=w=b_0\)
\(b_{i+1} = a_i - \lfloor \frac{a_i}{b_i} \rfloor b_i = p_{i-1}w + q_{i-1}v - z(p_iw + q_iv)= w(p_{i-1} - zp_i)+ v(q_{i-1} - zq_i)\) And so, \(p_{i+1}=p_{i-1}-zp_i\) and \(q_{i+1}=q_{i-1}-zq_i\). The existence of solution to this equation is called bezout's identity.

Extended Euclidean Algorithm

Input format: Two numbers \(w, v\), specifying values, for which extended euclid algorithm will be run.
\(w, v\)

First proof - Solution with different g

There are two important questions concerning extended gcd: firstly, whether is there any positive natural \(h < gcd(w,v)=g\) such that there exist integers \(x, y\) such that \(wx+vy=h\)? I'll prove opposite by contradiction:
\(wx+vy=h\)
\(gkx+gly=h\)
\(g(kx+ly)=h\)
 So, \(h\) must simultaneously fulfill: \(g|h\) and \(g > h > 0\). It is impossible, which finishes the proof. Furthermore, it shows, that for any \(x, y\) occurs \(gcd(w,v)|wx+vy\)

Second proof - different x, y

Secondly, how many there are solutions \(x, y\), fulfilling equation \(wx+vy=g\)? \(wx+vy=g\)
\(w(x+r_1)+v(y+r_2)=g\)
\(wx+wr_1+vy+vr_2=g\)
\(wr_1+vr_2=0\)
\(wr_1=-vr_2\)
 And so, there is infinite amount of possible solutions; it is obvious, if \(r_2=kw\), \(r_1=-kv\). But how to attain minimal next solution, i.e. \(s=|r_1w|\) is minimal for given \(r_1\)? Certainly:
\(w|s, v|s\) - it comes from last equation and fact, that \(r_1\) has to be integer - because by definition \(x+r_1\) is integer.
\(s > 0\) - otherwise \(r_1=0\) and so, next solution is still unknown.
\(s\) is minimal. In other words, \(s\) is least common multiple of \(w\) and \(v\) - or \(s=lcm(w, v)\).

One can show, that \(lcm(v, w)|vw\). By contradiction, suppose it is not true. What can be said about \(f=gcd(lcm(v,w), vw)\)?

  1. \(f \le lcm(v,w)\) - divisor of \(x\) must be lower than \(x\) (if x is positive).
  2. As \(v|lcm(v,w) \land w|lcm(v,w) \land w|wv \land v|wv\) , then, from euclid's algorithm, \(v|f \land w|f\)

So, if \(lcm(v, w) \nmid vw\), then \(gcd(lcm(v, w), vw)|vw\), it's divisible by both \(v, w\), and lower than \(lcm(v,w)\) - but it contradicts statement, that \(lcm(w, v)\) is lowest common mutiple of \(w, v\) - what follows, \(lcm(v,w)|vw\)

If \(s\) is shown as \(s=\frac{wv}{c}\), what can be said about \(c\)?
\(w|s \implies w|w\frac{v}{c} \implies c|v\)
\(v|s \implies v|v\frac{w}{c} \implies c|w\)
In order to minimize \(s=\frac{wv}{c}\), \(c\) has to be maximal possible. And maximal number \(c\) dividing both \(w\) and \(v\) is \(c=gcd(v, w)\). So, \(s=|rw|=\frac{wv}{gcd(w, v)}\). What follows, \(|r_1|=\frac{v}{gcd(w,v)}\) and \(|r_2|=\frac{w}{gcd(w,v)}\).

Third proof - estimation of \(a_i, b_i\)

First, one can see, that \(sgn(p_i) \neq sgn(p_{i+1})\) and \(sgn(q_i) \neq sgn(q_{i+1})\) - it is because:

  1. \(p_0=1, p_1=0\), also \(q_0=0, q_1=1\) - as inductive step is exactly same for \(p\) and \(q\), it'll be shown only for \(p\)
  2. By induction, \(p_i=p_{i-2}-zp_{i-1}\) - \(z\) is positive (as it is equal to \(\lfloor \frac{b_{i-2}}{b_{i-1}} \rfloor\)) - so it does not change sign, and as \(sgn(p_{i-1}) \neq sgn(p_{i-2})\), then:
    • If \(p_{i-2}\) is \(0\) - then \(p_{i-1}\) cannot be \(0\), and \(-zp_{i-1}\) have different sign than \(p_{i-1}\).
    • If \(p_{i-2}\) is not zero, then \(-zp_{i-1}\) either is equal to \(0\) - or \(-zp_{i-1}\) have same sign as \(p_{i-2}\), in both cases, \(sgn(p_{i-2})=sgn(p_i) \neq sgn(p_{i-1})\)
{% set z="\\lfloor \\frac{b_{i-1}}{b_i} \\rfloor " %}

One can also notice, that \(|p_{i-1}b_i-p_ib_{i-1}|=v\), and also \(|q_{i-1}b_i-q_ib_{i-1}|=w\) - proof is same for both, so inductive step will be shown only for \(p_i\) (1) \(|p_0b_1-p_1b_0|=|b_1-0|=v\), also \(|q_0b_1-q_1b_0|=|0-b_0|=|w|\) (2) \(|p_ib_{i+1}-p_{i+1}b_i|=|p_i(b_{i-1}-{{z}} b_i)-(p_{i-1}-{{z}} p_i)b_i|=|p_ib_{i-1}-{{z}} p_ib_i-p_{i-1}b_i+{{z}} p_ib_i|=|p_{i-1}b_i-p_ib_{i-1}|=v\)

These two lemmas show, that:

  1. Values of \(|p_i|, |q_i|\) for \(i \ge 2\) are not decreasing, as \(|p_i|=|p_{i-2}-{{z}}p_{i-1}|=|p_{i-2}|+|{{z}}p_{i-1}| \ge |p_{i-2}|+|p_{i-1}|\)
  2. \(|p_{i-1}b_i-p_ib_{i-1}|=|p_{i-1}b_i|+|p_ib_{i-1}|=v \implies |p_ib_{i-1}| \le v \implies |p_i| \le v\). Same holds for \(|q_i \le w\)

This shows, tha always \(p_i \le v\) and \(q_i \le w\)

Third Problem

Suppose you have to find all solutions \(x\) satisfying: \(ax \equiv c \pmod m\) for given \(1 \le a, c, m \le 10^9\)

Solution

One can show the problem as: \(\exists_{y \in Z} ax+my=c\) From Bezout's identity consequences, \(gcd(a, m)|c\) - otherwise, there are no solutions. If there are solutions however, then, from extended Euclidean algorithm, one can find such \(p, q \in Z\), such that: \(ap+mq=gcd(a, m) /// *\frac{c}{gcd(a, m)}\)
\(ap\frac{c}{gcd(a, m)}+mq\frac{c}{gcd(a, m)}=c\) and so, \(x=p\frac{c}{gcd(a, m)}\) solves this equation. Furthermore, next solutions have form \(x_k=(p+k\frac{m}{gcd(a,m)})\frac{c}{gcd(a, m)}\)

The notion of modular inverse

A number \(p\) is called a modular multiplicative inverse of \(a\) modulo \(m\) (where \(gcd(a, m)=1\)), if \(\exists_{q \in Z} ap+mq=1\) Such number \(p\) always exists - this is a consequence of extended Euclidean algorithm. One may note, that all modular inverses of \(a\) have same remainder from division by \(m\) - this comes from lemma above, as \(gcd(a, m)=1\) - and so, only numbers in form \(p+km\) can sitisfy this equation for some \(k \in Z\). The name comes from fact, that: \(ap \equiv 1 \pmod m\) Number \(p\) is often denoted as \(a^{-1}\).

Summary

  1. \((d|x \land d \ge 1) \implies (d \le \sqrt{x} \lor \frac{x}{d} \le \sqrt{x})\). Also, \(|\{d: d|x \land d \ge 1 \land x \ge 1\}| \le 2\sqrt{x}\)
  2. \(gcd(w, v)\) can be calculated in \(O(log(min(v,w)))\) with Euclidean algortithm, using fact, that \(gcd(w, v)=gcd(w\%v, v)\).
  3. One can find \(x, y\) solving equation \(wx+vy=gcd(w,v)\) also called bezout's identity using extended Euclidean algorithm. Also, for any \(x, y\) occurs \(gcd(w,v)|wx+vy\).
  4. \(lcm(v, w)=\frac{vw}{gcd(v,w)}\)
  5. If \((x,y)\) solves bezout's identity - \(wx+vy=gcd(w,v)\) -, then for any \(k \in Z\) pair \((x+\frac{kv}{gcd(w,v)}, y-\frac{kw}{gcd(w,v)})\) also solves this equation, and there are no other solutions.
  6. If \(gcd(m, a)=1\), then number \(p\) solving \(ap+mq=1\) is called modular multiplicative inverse of \(a\) modulo \(m\), also \(ap \equiv 1 \pmod m\)
