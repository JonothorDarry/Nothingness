Chinese Remainder Theorem

Problem statement and definitions

Suppose \(m\) equalities are given in form:
\( m\text{ equations} \begin{cases} {% for x in [1,2] %} x \equiv c_{{x}}\ (mod\ s_{{x}}) \\ {% endfor %} ... \\ x \equiv c_m\ (mod\ s_m) \\ \end{cases} \) The problem is: find \(x\) satisfying these equations

The first thesis

If it is possible to show equivalent equation for two different equations, then one can reduce number of equations by one:
\(x \equiv c_{1,2}\ (mod\ s_{1,2}) \iff \begin{cases} x \equiv c_1\ (mod\ s_1) \\ x \equiv c_2\ (mod\ s_2)\\ \end{cases} \) And, after reduction:

\( m-1\text{ equations} \begin{cases} x \equiv c_{1,2}\ (mod\ s_{1,2}) \\ x \equiv c_3\ (mod\ s_3) \\ ... \\ x \equiv c_m\ (mod\ s_m) \\ \end{cases} \) What follows, if one can solve pair of equations in \(O(w)\), then one can solve whole problem in \(O(w*m)\)

The second Thesis

One can rewrite
\( \begin{cases} x \equiv c_1\ (mod\ s_1) \\ x \equiv c_2\ (mod\ s_2)\\ \end{cases} \) To
\( \begin{cases} x = s_1a+c_1 \\ x = s_2b+c_2\\ \end{cases} \) For certain integers \(a,\ b\). What follows,
\(s_1a+c_1 = s_2b+c_2\) \(s_1a-s_1b=c_2-c_1\)
One can solve equation \(s_1k+s_2l=gcd(s_1, s_2)\) in \(O(log(min(s_1, s_2)))\) using extended euclid; If \(c_2-c_1 \nmid gcd(s_1,s_2)\), Then one can go out and solve different problems, for there are no integer solutions: left side is always divsible by \(gcd(s_1,s_2)\) when \(s_1\) and \(s_2\) are multiplied by integers, \(c2-c1\) isn't. However, if \((c_2-c_1)|gcd(s_1,s_2)\), then:
\(s_1k+s_2l = gcd(s_1,s_2)\ \ \ /*\frac{c_2-c_1}{gcd(s_1,s_2)}\) \(s_1k*\frac{c_2-c_1}{gcd(s_1,s_2)}+s_2l*\frac{c_2-c_1}{gcd(s_1,s_2)} = c_2-c_1\)
And so, \(a=k\frac{c_2-c_1}{gcd(s_1,s_2)},\ b=l\frac{c_2-c_1}{gcd(s_1,s_2)}\). Now, \(x = s_1a+c_1 = s_1k\frac{c_2-c_1}{gcd(s_1,s_2)}+c_1\)
Which finishes stated subproblem in \(O(log(max(s_i)))\), and whole problem in \(O(m*log(max(s_i)))\)
